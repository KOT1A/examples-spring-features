# rate-limiter
Пример Rate Limiter в Spring boot

___

## Краткое описание что такое Rate Limiter
Rate Limiter ограничивает количество операций (обычно запросов) за единицу времени.
Например: не больше 100 запросов в секунду от одного клиента или не больше 10 операций на конкретный endpoint.<br>
Цель простая:
защитить сервис от перегрузки и сдерживать агрессивных клиентов, чтобы твой сервис работал стабильно.
